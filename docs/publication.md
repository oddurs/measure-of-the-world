# Pre-Publication Checklist

## Legal & Licensing

- [ ] **ISBN** - Get an ISBN (can use free ones from some self-publishing platforms, or purchase from Bowker)
- [ ] **License** - For open source books, consider:
  - **CC BY 4.0** (Creative Commons Attribution) - allows commercial use with attribution
  - **CC BY-SA 4.0** - same but derivatives must use same license
  - **CC BY-NC 4.0** - non-commercial use only
- [ ] **Add LICENSE file** to GitHub repo
- [ ] **Reference license in book** - typically on copyright page

## Content Completion

- [ ] Table of Contents finalized
- [ ] All chapters complete
- [ ] Bibliography/References
- [ ] Index (for print)
- [ ] Acknowledgments
- [ ] About the Author
- [ ] Preface/Introduction

## Quality Checks

- [ ] Proofreading/editing pass
- [ ] Fact-checking
- [ ] Image rights cleared
- [ ] Code samples tested
- [ ] Consistent formatting

## Distribution

- [ ] Choose platforms (Amazon KDP, Lulu, Gumroad, etc.)
- [ ] Cover design (front, spine, back for print)
- [ ] Metadata (description, categories, keywords)
- [ ] Pricing strategy (free PDF, paid print?)
